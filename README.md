# Mục tiêu
---
- **Xử lý lỗi tập trung và Truy vấn dữ liệu**
- **Công nghệ sử dụng: Java 17, Spring Boot, Spring Data JPA, MySQL, Lombok, Maven**
---
# Danh sách API
- **POST /api/students: Thêm mới sinh viên**
- **GET /api/students/{id}: Lấy sinh viên theo ID (Xử lý lỗi)**
- **GET /api/students/search: Tìm kiếm theo tên (Custom Query)**
